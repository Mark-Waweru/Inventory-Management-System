# BigShelf Inventory Management System
The Inventory Management System is a web-based application built with Ruby on Rails and Hotwire to help administrators manage products and track stock levels in a retail store.
The system provides real-time stock updates, low-stock notifications, and an intuitive user interface for managing inventory efficiently.

## Getting Started (Step-by-Step)
Follow this steps to run the Application Locally.

**Step 1: Clone the Repository**
```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```
**Step 2: Install Ruby Dependancies**<br>
Install all required gems using bundler
```bash
bundle install
```
**Step 3: Setup the Database**<br>
Create and migrate the database
```bash
bin/rails db:create
bin/rails db:migrate
bin/rails db:seed
```
**Step 4: Start the Rails Server**<br>
Run the development server
```bash
bin/rails server
```
Open your browser and visit http://localhost:3000. You should see the Rails Application Running locally.<br>
